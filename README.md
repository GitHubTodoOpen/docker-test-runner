# Docker image for running tests

Docker for testing and build projects at [iGLOO](http://igloo.be).
Available on Docker Hub: `igloo/test-runner`

## Features

- git
- curl
- wget
- rsync
- patch
- build-essential
- nvm
- rvm
- python
- mysql-client-5.6
- phantomjs dependencies (libfreetype6 libfontconfig)
- wkhtmltopdf 0.12.2
- default-jre
